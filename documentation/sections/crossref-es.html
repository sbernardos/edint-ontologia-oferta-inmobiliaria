Anuncio Inmobiliarioc Volver a índice o Índice de clases

IRI: http://vocab.linkeddata.es/datosabiertos/def/oferta-inmobiliaria/RealEstateListing

Un anuncio inmobiliario que representa una oferta de un inmueble (venta, alquiler o traspaso). Cada anuncio tiene un identificador único y describe el tipo de inmueble, tipo de transacción y estado actual.

se refiere a entidad catastralop Volver a índice o Índice de propiedades de objetos

IRI: http://vocab.linkeddata.es/datosabiertos/def/oferta-inmobiliaria/refersToCadastralEntity

Relación que vincula un anuncio inmobiliario con su entidad catastral correspondiente (parcela o inmueble). Esto permite acceder a información catastral como el uso, la clase, la dirección y la referencia a través de la ontología del Catastro. Para ventas de solo terreno, se enlaza a una CadastralParcel. Para edificios o unidades, se enlaza a una CadastralProperty.
