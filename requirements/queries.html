CQ01: Transaction Types / Tipos de Transacción

Question / Pregunta: What types of real estate transactions exist in the listings? / ¿Qué tipos de transacción inmobiliaria existen en los anuncios?

Description / Descripción: Retrieve all distinct transaction types available in the ontology / Recuperar todos los tipos de transacción distintos disponibles en la ontología

High PriorityClass: RealEstateListingProperty: transactionType

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX skos: <http://www.w3.org/2004/02/skos/core#> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT DISTINCT ?transactionType ?transactionLabel ?transactionLabelEs WHERE { ?listing a edintinm:RealEstateListing ; edintinm:transactionType ?transactionType . OPTIONAL { ?transactionType rdfs:label ?transactionLabel . FILTER (lang(?transactionLabel) = "en") } OPTIONAL { ?transactionType rdfs:label ?transactionLabelEs . FILTER (lang(?transactionLabelEs) = "es") } } ORDER BY ?transactionLabel

CQ02: Listing Information / Información del Anuncio

Question / Pregunta: What information is contained in a real estate listing? / ¿Qué información contiene un anuncio inmobiliario?

Description / Descripción: Query all properties and attributes of real estate listings / Consultar todas las propiedades y atributos de los anuncios inmobiliarios

High PriorityClass: RealEstateListingMultiple Properties

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX dcterms: <http://purl.org/dc/terms/> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT ?listing ?listingId ?isActive ?propertyTypeLabelEs ?transactionTypeLabelEs WHERE { ?listing a edintinm:RealEstateListing ; dcterms:identifier ?listingId ; edintinm:isActive ?isActive . OPTIONAL { ?listing edintinm:propertyType ?propertyType . ?propertyType rdfs:label ?propertyTypeLabelEs . FILTER (lang(?propertyTypeLabelEs) = "es") } OPTIONAL { ?listing edintinm:transactionType ?transactionType . ?transactionType rdfs:label ?transactionTypeLabelEs . FILTER (lang(?transactionTypeLabelEs) = "es") } } ORDER BY ?listingId

CQ03: Cadastral Relationships / Relaciones Catastrales

Question / Pregunta: How is a listing related to a cadastral entity? / ¿Cómo se relaciona un anuncio con una entidad catastral?

Description / Descripción: Find cadastral entities referenced by real estate listings / Encontrar entidades catastrales referenciadas por los anuncios inmobiliarios

High PriorityClass: RealEstateListingProperty: refersToCadastralEntity

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX dcterms: <http://purl.org/dc/terms/> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT ?listing ?listingId ?cadastralEntity ?entityType ?entityLabelEs WHERE { ?listing a edintinm:RealEstateListing ; dcterms:identifier ?listingId ; edintinm:refersToCadastralEntity ?cadastralEntity . ?cadastralEntity rdfs:label ?entityLabelEs . FILTER (lang(?entityLabelEs) = "es") BIND( IF(STRENDS(STR(?cadastralEntity), "CadastralParcel"), "Parcela Catastral", IF(STRENDS(STR(?cadastralEntity), "CadastralProperty"), "Inmueble Catastral", "Entidad Desconocida") ) AS ?entityType) } ORDER BY ?listingId

CQ04: Sale Properties / Propiedades en Venta

Question / Pregunta: What properties does a property have in a sale listing? / ¿Qué propiedades tiene un inmueble en venta?

Description / Descripción: Query properties with specific transaction type (sale) / Consultar propiedades con tipo de transacción específico (venta)

High PriorityClass: RealEstateListingProperty: propertyType, transactionType

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX skos: <http://www.w3.org/2004/02/skos/core#> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT DISTINCT ?listing ?listingId ?propertyType ?propertyLabelEs ?propertyTypeLabelEs WHERE { ?listing a edintinm:RealEstateListing ; edintinm:propertyType ?propertyType ; edintinm:transactionType ?transactionType . ?transactionType skos:prefLabel ?transactionLabel . FILTER (STRLCASE(STR(?transactionLabel)) = "venta" || STRLCASE(STR(?transactionLabel)) = "sale") OPTIONAL { ?propertyType rdfs:label ?propertyLabelEs . FILTER (lang(?propertyLabelEs) = "es") } OPTIONAL { ?propertyType skos:prefLabel ?propertyTypeLabelEs . FILTER (lang(?propertyTypeLabelEs) = "es") } } ORDER BY ?listingId

CQ05: Listing Status / Estado del Anuncio

Question / Pregunta: What is the current status of a real estate listing? / ¿Cuál es el estado actual de un anuncio inmobiliario?

Description / Descripción: Check status of real estate listings (active/inactive) / Verificar estado de los anuncios inmobiliarios (activo/inactivo)

High PriorityClass: RealEstateListingProperty: isActive

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX dcterms: <http://purl.org/dc/terms/> SELECT ?listing ?listingId ?isActive ?statusLabel WHERE { ?listing a edintinm:RealEstateListing ; dcterms:identifier ?listingId ; edintinm:isActive ?isActive . BIND( IF(?isActive = true, "Activo"^^<http://www.w3.org/2001/XMLSchema#string>, "Inactivo"^^<http://www.w3.org/2001/XMLSchema#string>) AS ?statusLabel) } ORDER BY ?listingId

CQ07: Location Specification / Especificación de Ubicación

Question / Pregunta: How is the location of a property specified? / ¿Cómo se especifica la ubicación de un inmueble?

Description / Descripción: Query geographic location of properties in listings / Consultar ubicación geográfica de los inmuebles en los anuncios

High PriorityClass: RealEstateListingProperty: refersToCadastralEntity

SPARQL Query:

PREFIX edintinm: <https://edint.github.io/edint-ontologia-inmuebles/ontology/inmobiliaria#> PREFIX dcterms: <http://purl.org/dc/terms/> PREFIX geo: <http://www.opengis.net/ont/geosparql#> PREFIX catastro: <https://edint.github.io/edint-ontologia-catastro/ontology/catastro#> SELECT ?listing ?listingId ?cadastralEntity ?hasGeometry ?wktCoordinate ?address WHERE { ?listing a edintinm:RealEstateListing ; dcterms:identifier ?listingId ; edintinm:refersToCadastralEntity ?cadastralEntity . OPTIONAL { ?cadastralEntity geo:hasGeometry ?hasGeometry . OPTIONAL { ?hasGeometry geo:asWKT ?wktCoordinate . } } OPTIONAL { ?cadastralEntity catastro:hasAddress ?address . } } ORDER BY ?listingId
